# BSTM Marketplace - Room 18

Africa's First Digital Nation Marketplace

## 🌟 Features
- Multi-vendor marketplace
- Paystack payment integration
- THoBoCoin (THB) rewards
- Room 22 Farm integration
- CabLink delivery integration
- PWA support (offline capable)
- Real-time updates via Supabase

## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- npm or yarn
- Supabase account
- Paystack account

http://localhost:3000

/frontend - React application
/database - Supabase schema
